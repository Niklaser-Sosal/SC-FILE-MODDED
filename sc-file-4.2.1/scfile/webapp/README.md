# sc-file Web UI

Локальный web-интерфейс для `sc-file` с:

- пакетной конвертацией (папка или ZIP)
- логированием
- настройками/темой (несколько тем + кастомизация)
- фоном (градиент/картинка), уведомлениями, RU/EN
- вкладкой Map Merge (на базе `sc-mapmerge`)

## Запуск

### В этой папке (Windows)

Запусти `scfile-web.bat` из корня (рядом со `sc-file-*` и `sc-mapmerge-*`).

UI-ассеты лежат в `webapp/static` (в корне репозитория) и могут свободно редактироваться.

### Через pip

Примечание: в этой папочной версии UI-ассеты вынесены в `webapp/static` (в корне). Если ты ставишь пакет через pip,
убедись, что ассеты доступны (или используй запуск из этой папки через `scfile-web.bat`).

```bash
pip install "sc-file[web]"
scfile --web
```

Или напрямую:

```bash
python -m scfile.webapp
```

### Без окна (только браузер)

```bash
scfile --web --no-window
```

## Map Merge

Для вкладки Map Merge нужен модуль `scmapmerge` (проект `sc-mapmerge`) рядом с `sc-file` в этой папке
или установленный как пакет.
